# LuxEstate

Lüks yaşamı merkezine alan, Next.js tabanlı etkileşimli emlak deneyimi.

## Özellikler
- App Router ile kurulu Next.js 14 mimarisi
- Özenle tasarlanmış hero, portföy, deneyim ve concierge bölümleri
- Tıklanabilir mülk kartları ve her mülk için detay sayfası
- Uzaktan erişilebilen yüksek çözünürlüklü galeri ve ayrıcalık listeleri
- Mobil uyumlu, modern ve zarif tasarım

## Geliştirme
```bash
npm install
npm run dev
```

Uygulama varsayılan olarak [http://localhost:3000](http://localhost:3000) adresinde çalışır.

## Üretim
```bash
npm run build
npm start
```
